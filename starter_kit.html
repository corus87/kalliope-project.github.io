Starter kits

A starter kit is a repository that contains an already made configuration with some brain examples for a language. Starter kit are a good entrypoint to learn how Kalliope works.

How to use a starter kit:

  • Follow the doc to install Kalliope
  • Clone a starter kit: git clone <starter_kit_git_url.git>
  • Place yourself into the starter kit: cd cloned_starter_kit
  • Start kalliope: kalliope start
